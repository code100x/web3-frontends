## Environment Setup

Before using the `NftCard` component, ensure you have the `NEXT_PUBLIC_NFTSCAN_KEY` environment variable set in your project. This key is necessary for fetching NFT data from the NFTScan API.

Create or update your `.env` file in the root of your project with the following line:

```env
NEXT_PUBLIC_NFTSCAN_KEY=your_nftscan_api_key_here
```

Replace `your_nftscan_api_key_here` with your actual NFTScan API key. You can obtain this key by registering and creating an API key on the [NFTScan developers portal](https://developer.nftscan.com/).

## Usage

First, import the `NftCard` component:

```tsx
import { NftCard } from "@/components/ui/nft-card"
```

Then, use it in your project by passing the required props, like so:

```tsx
<NftCard
  mintAddress="YourNFTMintAddress"
  variant="dark"
  size="lg"
  imgRatio="square"
  nftName="visible"
  collectionName="visible"
/>
```
